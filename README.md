Introduction to Programming Project
===================================
Image editing
-------------

#### Features: ####
* Grey tones
* Change basic colors
* Horizontal and vertical reflections
* Negative image
* Rotations
* Cutting and Bordering
* Hide and read hidden messages
* Encrypt and decrypt images

####Usage:####

    python imaging.py
(requires python 2.* and PIL)
